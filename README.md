# Pulkit Sinha — Portfolio

Personal portfolio website. Built with plain HTML/CSS/JS — no framework, no build step.

**Live:** [pulkit-sinha.com](https://pulkit-sinha.com)

## Tech Stack

- HTML / CSS / JS (ES modules, no bundler)
- [GSAP 3](https://gsap.com) + ScrollTrigger — animations
- [Lenis](https://github.com/darkroomengineering/lenis) — smooth scroll
- [Three.js](https://threejs.org) — 3D katana, mask, and asterisk elements
- [Lucide](https://lucide.dev) — icons
- Hosted on Cloudflare Pages

## Development

```bash
python3 -m http.server 3333
# Open http://localhost:3333
```
